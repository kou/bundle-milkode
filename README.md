# bundle-milkode

bundle-milkode makes gems installed by Bundler milkable automatically.

## Install

    % gem install bundle-milkode

## How to use

Use `bundle-milkode` instead of `bundle`:

    % bundle-milkode install
    % bundle-milkode update
